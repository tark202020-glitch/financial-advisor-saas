### 1. 🚨 [단일 통화 통합] 포트폴리오 원화 합산 로직 전면 개편 (가장 많은 수정 요청 사항)
- **증상**: '내 주식 종합' 화면에서 국내 자산(원)과 해외 자산(달러) 보유 시, "100달러 + 100원 = 200"과 같이 단위 구분 없이 단순 합산되는 심각한 오류가 있었습니다.
- **해결 및 대응**: 
  - `PortfolioContext`에 실시간 원/달러 환율(`API(/api/market-extra)`) 데이터를 전역 로드하는 로직 신설.
  - `PortfolioSummaryBlock` 내의 '전체' 뷰, 카테고리별 요약, 실현 손익 등 모든 자산 통계를 **한화(KRW) 기준**으로 계산하여 적용하는 로직 구현.
  - 필터 탭('해외' 등) 근처에 명시적으로 적용되는 "적용 환율: $1,350원" 안내 UI 부착.
  - **수차례 반복 보완된 디테일**: "모든 금액을 한화로 표현해야 한다"는 요청에 따라, `view` 조건과 상관없이 원화 단위(`원`) 포매팅을 강제로 적용하여 통화 혼동을 근절시켰습니다.

### 2. KIS 현재가 조회 모듈 (NXT) 개선
- **증상**: KIS API에서 정규장(KRX)의 체결가와 실제 현재 시세 간극이 크게 남.
- **해결**: 가격 조회 요청(`getDomesticPrice`)의 파라미터를 'J'(KRX)에서 'NX'(NXT)로 교체하여 신뢰도와 데이터 정합성을 한층 증가시켰습니다.

### 3. 주식 스터디(로컬 전용 게시판) 오픈 및 코스피/MSCI 등재
- **주요 구현**: '조건 검색' 기능을 폐기하고, 로컬 마크다운(`.md`) 파일들을 연동 및 실시간 수정/조회 가능한 '주식 스터디' 공간 구현 완료 (`doc/MSCI` 등 반영).
- **랜더링 고도화**: 커스텀 Markdown 파서를 제작해 파단 표기(`| 표 |`) 문법이 시각적인 테이블 레이아웃으로 변환되도록 구축. 또한 코스피 장외/미응답 에러 방어(Fallback) 메인 로직을 탑재했습니다.

# Daily Report - 2026-02-18

## 📅 주요 작업 내역

### 1. 일일체크 데이터 정확도 개선 (V1.237)
- **증상**: 일일체크 화면에서 환율 변동폭이 0으로 표시되고, 금값이 달러 단위라 직관적이지 않으며, 금리가 최신화되지 않음.
- **해결**:
  - **환율**: Yahoo Finance API의 `chartPreviousClose` 데이터를 활용하여 전일 대비 등락폭(Change/%)이 0으로 나오는 문제 해결.
  - **금**: 기존 USD/oz 단위를 **KRW/g** 단위로 자동 환산하여 표기 (환율 연동).
  - **금리**: 미국 기준금리를 최신 데이터 범위(**3.50% ~ 3.75%**)로 현행화.
  - **기준일**: 모든 시장 지표에 데이터 기준 시간(Reference Date)을 명시하여 신뢰도 향상.

## 📝 내일의 계획
- **모니터링**: 배포 후 일일체크 데이터가 장중/장마감 시점에 올바르게 갱신되는지 확인.
- **추가 개선**: 필요 시 금리 데이터를 실시간 API(FRED 등) 연동으로 고도화 검토.
