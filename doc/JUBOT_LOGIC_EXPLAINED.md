# 🤖 JUBOT 분석 로직 및 데이터 출처 상세

주봇(Jubot)이 제공하는 분석 정보가 어떤 데이터를 기반으로 생성되는지, 그리고 어떤 기준으로 분석하는지 상세 내용을 정리합니다.

---

## 1. 🌅 일일 체크 (시장 브리핑)

매일 아침 시장의 흐름을 파악하여 브리핑해주는 기능입니다.

### 📊 데이터 출처 (Data Sources)
1.  **뉴스 데이터 (News RSS)** &nbsp; `실시간`
    *   **출처**: 매일경제 증권 뉴스, 연합뉴스 경제 뉴스
    *   **수집량**: 최신 뉴스 **상위 15개**를 수집하여 AI가 중요도를 판단합니다.
2.  **시장 지수 (Market Indices)** &nbsp; `지연/실시간`
    *   **출처**: KIS(한국투자증권) API
    *   **항목**: 코스피(KOSPI), 코스닥(KOSDAQ) 등의 현재 지수 및 등락률.

### 🧠 분석 로직 (Analysis Logic)
*   **AI 모델**: Google Gemini 2.5 Flash
*   **생성 시간**: 매일 오전 07:00 (한국시간 기준)
*   **캐싱(Caching)**: 하루에 한 번만 생성되며, 이후 재방문 시에는 **DB에 저장된 브리핑**을 즉시 불러옵니다.
*   **핵심 기준**: 단순히 뉴스를 나열하는 것이 아니라, 수집된 15개의 뉴스 중 **시장에 미칠 파급력이 큰 이슈 3~5개**를 선별하여 요약합니다.

---

## 2. 💼 내 주식 인사이트 (포트폴리오 분석)

보유한 전체 주식을 종합적으로 분석하여 어떤 대응이 필요한지 우선순위를 정해주는 기능입니다.

### 📊 데이터 출처
1.  **보유 종목 정보**: 사용자가 입력한 평단가, 보유수량, 수익률.
2.  **재무 데이터 (Financials)**: **OpenDART(금융감독원)** API
    *   매출액, 영업이익, 당기순이익 등 핵심 지표.
3.  **공시 정보 (Disclosures)**: **OpenDART** API
    *   최근 6개월 간의 주요 공시 (사업보고서, 실적발표, 유상증자 등).
4.  **배당 정보 (Dividends)**: **OpenDART** API
    *   최근 배당금(DPS), 배당성향.

### 🧠 분석 우선순위 (Priority Logic)
Jubot은 단순 수익률 등락보다 **실질적인 이벤트**를 중요하게 여깁니다. 다음 순서대로 우선순위를 두어 조언합니다.

1.  🥇 **제 1순위: 공시(Disclosure) 분석**
    *   주가에 급격한 영향을 줄 수 있는 공시(예: 어닝 쇼크, 대규모 수주, 유상증자)가 발견되면 가장 먼저 경고합니다.
2.  🥈 **제 2순위: 배당(Dividend) 체크**
    *   예상 배당금 총액이 **1만원 이상**인 경우, 배당 기준일과 예상 수령액을 안내합니다.
3.  🥉 **제 3순위: 구체적 액션(Action)**
    *   다가오는 일정(실적발표일, 주총 등)이 있을 때 무엇을 확인해야 하는지 알려줍니다.
4.  **기타**: 단순 가격 변동이나 수익률에 대한 코멘트.

## 3. 🔍 종목 심층 분석 (Stock Detail)

특정 종목 하나를 클릭했을 때 제공되는 상세 분석입니다.

### 📊 데이터 출처
1.  **실시간 시세**: **KIS API**를 통해 조회한 현재가 (해외주식 포함).
2.  **3개년 재무제표**: OpenDART에서 가져온 **최근 3년**간의 매출/영업이익 추이.
3.  **관련 뉴스**: 해당 **종목명**이 제목에 포함된 최신 뉴스 **상위 5개** (네이버/한경 등 RSS 파싱).

### 🧠 분석 기준
*   **종합 신호 (Signal)**: `매수(Buy)` / `보유(Hold)` / `매도(Sell)` / `관망(Watch)` 4단계로 제시.
    *   *주의: 이는 AI의 분석 의견일 뿐, 절대적인 투자 권유가 아닙니다.*
*   **재무 건전성**: 영업이익이 꾸준히 증가하는지, 적자가 지속되는지 평가.
*   **뉴스 영향도**: 최근 뉴스가 주가에 긍정적(호재)인지 부정적(악재)인지 판별.
*   **목표가 달성 여부**: 사용자가 설정한 목표가(상/하한)에 근접했는지 여부.

### 📝 요약

| 구분 | 주요 데이터 소스 | 갱신 주기 | 핵심 특징 |
| :--- | :--- | :--- | :--- |
| **일일 브리핑** | 뉴스 RSS (15개), 시장지수 | 매일 아침 (1회) | 시장 전반의 핵심 이슈 요약 |
| **포트폴리오** | OpenDART (재무/공시/배당) | 페이지 방문 시 | 공시/이벤트 위주의 **우선순위** 조언 |
| **종목 상세** | KIS (현재가), OpenDART (3년치) | 모달 열 때 (캐싱됨) | 재무+뉴스+가격을 종합한 **심층 분석** |

> **참고**: 모든 분석 데이터는 생성 시점(Timestamp)이 기록되며, 분석 후 시간이 지나면 재분석 버튼을 통해 갱신할 수 있습니다.
