# 🤖 JUBOT (주봇) — 분석 로직 및 데이터 출처

> **현재 버전: 주봇 1.0**

주봇(JUBOT)이 제공하는 분석 정보가 어떤 데이터를 기반으로 생성되는지, 그리고 어떤 기준으로 분석하는지 상세 내용을 정리합니다.

---

## 버전 이력

| 버전 | 날짜 | 주요 변경 |
| :--- | :--- | :--- |
| **주봇 1.0** | 2026-02-14 | 뉴스 소스 확대(4개), 전문가 우선 노출, 거래기록 리뷰(0순위), 포트폴리오에 뉴스 통합 |
| 주봇 0.9 (초기) | 2026-01 ~ 02 | 기본 뉴스 수집(2개), 재무/공시/배당 기반 분석, 일일 브리핑 |

---

## 1. 🌅 일일 체크 (시장 브리핑)

매일 아침 시장의 흐름을 파악하여 브리핑해주는 기능입니다.

### 📊 데이터 출처 (Data Sources)
1. **뉴스 데이터 (News RSS)** &nbsp; `실시간`
    * **출처** (주봇 1.0):
        * 매일경제 증권 뉴스
        * 연합뉴스 경제 뉴스
        * 🆕 네이버 경제 뉴스
        * 🆕 인베스팅닷컴 (Investing.com) 한국
    * **수집량**: 소스별 최대 **10개**, 총 최대 **40개** 기사 수집
2. **시장 지수 (Market Indices)** &nbsp; `지연/실시간`
    * **출처**: Yahoo Finance API
    * **항목**: KOSPI, KOSDAQ, S&P500, NASDAQ 지수 및 등락률
3. **환율/금/금리** &nbsp; `실시간`
    * USD/KRW, JPY/KRW, CNY/KRW, 금 시세, 한국/미국 기준금리

### 🎤 전문가 우선 노출 (주봇 1.0)
* **대상 전문가**: 박시동, 이광수
* **동작**: 수집된 뉴스 중 위 전문가가 언급된 기사를 자동 감지하여 `expert_opinions` 섹션에 최우선 배치
* **표시**: 브리핑 UI에 🎤 전문가 의견 섹션으로 별도 표시

### 🧠 분석 로직 (Analysis Logic)
* **AI 모델**: Google Gemini 2.5 Flash
* **생성 시간**: 매일 오전 07:00 (한국시간 기준)
* **캐싱**: DB 캐시 + SessionStorage (즉시 로딩)
* **핵심 기준**: 수집된 뉴스 중 **시장에 미칠 파급력이 큰 이슈 3~5개**를 선별하여 요약

## 2. 💼 내 주식 인사이트 (포트폴리오 분석)

보유한 전체 주식을 종합적으로 분석하여 어떤 대응이 필요한지 우선순위를 정해주는 기능입니다.

### 📊 데이터 출처
1. **보유 종목 정보**: 사용자가 입력한 평단가, 보유수량, 수익률
2. **거래기록 (Trade History)** 🆕: 사용자의 매수/매도 이력
3. **재무 데이터 (Financials)**: **OpenDART(금융감독원)** API
    * 매출액, 영업이익, 당기순이익 등 핵심 지표
4. **공시 정보 (Disclosures)**: **OpenDART** API
    * 최근 6개월 간의 주요 공시
5. **배당 정보 (Dividends)**: **OpenDART** API
    * 최근 배당금(DPS), 배당성향
6. **뉴스 데이터** 🆕: 일일 체크와 동일한 4개 소스
    * 매일경제 증권, 연합뉴스 경제, 네이버 경제, 인베스팅닷컴
    * 전문가(박시동, 이광수) 관련 기사 우선 반영

### 🧠 분석 우선순위 (Priority Logic)

주봇 1.0은 **거래기록 리뷰**를 최우선으로 분석합니다.

| 순위 | 항목 | 설명 |
| :--- | :--- | :--- |
| 🏆 **0순위** | **거래기록 리뷰** | 최근 매수/매도 기록에 대한 타이밍 평가와 수익률 해석 |
| 🥇 1순위 | 공시(Disclosure) 분석 | 주가에 급격한 영향을 줄 수 있는 공시 경고 |
| 🥈 2순위 | 배당(Dividend) 체크 | 예상 배당금 총액 ≥ 1만원일 때 일정/금액 안내 |
| 🥉 3순위 | 구체적 액션(Action) | 다가오는 일정(실적발표일, 주총 등) 안내 |
| ⬜ 4순위 | 가격/수익률 | 단순 가격 변동, 목표가 근접 여부 |

### 🎤 전문가 우선 노출 (주봇 1.0)
* 뉴스 중 **박시동, 이광수** 관련 기사가 있으면 해당 종목의 `related_news`에 우선 반영

## 3. 🔍 종목 심층 분석 (Stock Detail)

특정 종목 하나를 클릭했을 때 제공되는 상세 분석입니다.

### 📊 데이터 출처
1. **실시간 시세**: **KIS API**를 통해 조회한 현재가 (해외주식 포함)
2. **3개년 재무제표**: OpenDART에서 가져온 **최근 3년**간의 매출/영업이익 추이
3. **관련 뉴스**: 해당 **종목명**이 제목에 포함된 최신 뉴스 **상위 5개**

### 🧠 분석 기준
* **종합 신호 (Signal)**: `매수(Buy)` / `보유(Hold)` / `매도(Sell)` / `관망(Watch)` 4단계
* **재무 건전성**: 영업이익 추이, 적자 지속 여부
* **뉴스 영향도**: 호재/악재 판별
* **목표가 달성 여부**: 사용자 설정 목표가(상/하한) 근접 여부

### 📝 요약

| 구분 | 주요 데이터 소스 | 갱신 주기 | 핵심 특징 |
| :--- | :--- | :--- | :--- |
| **일일 브리핑** | 뉴스 RSS (4개 소스, 최대 40개), 시장지수 | 매일 아침 (1회) | 시장 이슈 요약 + **전문가 의견** |
| **포트폴리오** | OpenDART + 뉴스 + 거래기록 | 페이지 방문 시 | **거래 리뷰(0순위)** + 공시/배당 조언 |
| **종목 상세** | KIS (현재가), OpenDART (3년치), 뉴스 | 모달 열 때 (캐싱) | 재무+뉴스+가격 **심층 분석** |

> **참고**: 모든 분석 데이터는 생성 시점(Timestamp)이 기록되며, 분석 후 시간이 지나면 재분석 버튼을 통해 갱신할 수 있습니다.
