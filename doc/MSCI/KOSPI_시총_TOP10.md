### 📌 참고사항

- 시가총액 데이터는 **대략적인 추정치**이며, 실시간 정확한 데이터는 장중(09:00~15:30 KST)에 KIS API를 통해 조회해야 합니다.
- KIS API 시총 순위 (TR_ID: `FHPST01730000`)는 장중에만 데이터를 반환합니다.
- 장외 시간에는 개별 종목 현재가 API (TR_ID: `FHKST01010100`)의 `hts_avls` 필드에서 시가총액(억원 단위)을 조회할 수 있습니다.
- 비중은 코스피 전체 시가총액 대비 각 종목의 비율입니다.

### 🔧 실시간 데이터 조회 방법

프로젝트 내 스크립트를 사용하여 실시간 데이터를 조회할 수 있습니다:

```bash
# Dev 서버 실행 후
npm run dev

# 시총 순위 API 호출 (장중에만 유효)
curl http://localhost:3000/api/kis/ranking?type=market-cap
```

또는 독립 스크립트 실행:

```bash
node scripts/fetch_kospi_top10.mjs
```
