## 2. 배당주 TOP10 분석 로직

> 소스: `src/app/api/study/generate-dividend/route.ts`

### 2-1. 처리 흐름도

```
[1단계] KIS 배당률 상위 순위 조회 (코스피 보통주, 현금배당)
    ↓ 상위 50개 후보 확보
[2단계] 후보 50개를 순차 처리 (10개 확보 시 조기 종료)
    ├─ 종목명 조회 (getStockInfo)
    ├─ 실제 배당 이력 조회 (getKsdinfoDividend, 최근 2년)
    ├─ 현재가(종가) 조회 (getDomesticPrice)
    └─ 수익률 및 가상배당금 계산
    ↓ 수익률 내림차순 정렬
[3단계] 마크다운 테이블 생성
[4단계] Supabase 저장
```

### 2-2. 단계별 상세 설명

#### 1단계: 배당률 상위 순위 조회

```
API: getDividendRateRanking()
KIS TR_ID: HHKDB13470100
URL: /uapi/domestic-stock/v1/ranking/dividend-rate
```

| 파라미터 | 값 | 설명 |
|---------|---|------|
| `GB1` | `1` | 코스피만 |
| `UPJONG` | `0001` | 종합 (전체 업종) |
| `GB2` | `6` | 보통주만 (우선주 제외) |
| `GB3` | `2` | 현금배당만 |
| `F_DT` | 1년 전 | 순위 조회 시작일 |
| `T_DT` | 오늘 | 순위 조회 종료일 |
| `GB4` | `0` | 결산+중간배당 전체 |

→ 결과에서 **상위 50개** 후보를 추출

#### 2단계: 후보 종목 검증 (순차 처리)

50개 후보를 1건씩 순차 처리하며, **유효한 10건**이 확보되면 즉시 종료합니다.

각 종목별 처리 과정:

1. **종목명 조회** — `getStockInfo(code)` → `prdt_abrv_name`
2. **실제 배당 이력 조회** — `getKsdinfoDividend()` (최근 2년, 해당 종목)
   - `per_sto_divi_amt > 0`인 레코드만 필터링
   - `record_date` 기준 내림차순 정렬 → **가장 최근 배당** 선택
3. **현재가 조회** — `getDomesticPrice(code)` → `stck_prpr`
4. **수익률 계산**:
   ```
   수익률(%) = (주당배당금 / 현재 종가) × 100
   가상배당금 = floor(1,000만원 / 종가) × 주당배당금
   ```
5. API Rate Limit 대응: 각 호출 사이 200ms 대기

#### 3단계: 마크다운 리포트 생성

| 컬럼 | 내용 |
|------|------|
| 종목 | 종목 한글명 |
| 종가 | 현재 종가 (원) |
| 주당배당금 | 실제 지급 금액 + 지급일 |
| 수익률 | 종가 대비 배당수익률 (%) |
| 최근배당일 | 기준일(record_date) |
| 가상배당금 | 1,000만원 투자 시 예상 배당금 |

## 3. 배당 ETF TOP10 분석 로직

> 소스: `src/app/api/study/generate-dividend-etf/route.ts`

### 3-1. 처리 흐름도

```
[1단계] 마스터 JSON에서 ETF 후보군 동적 파싱 (키워드 필터링)
    ↓ 배당/액티브/보험/은행 키워드 + 커버드콜 제외 + ETF 브랜드 확인
[2단계] ETF 현재가 조회 (15개씩 청크 병렬 처리)
    ↓ 유효 ETF 확보
[3단계] 실제 배당 이력 조회 (15개씩 청크 병렬 처리)
    ↓ 수익률 내림차순 정렬 → TOP 10 추출
[4단계] 마크다운 테이블 생성
[5단계] Supabase 저장
```

### 3-2. 단계별 상세 설명

#### 1단계: ETF 후보군 동적 파싱

데이터 소스: `public/data/all_stocks_master.json` (전 종목 마스터)

**3중 필터링 구조:**

| 순서 | 필터 | 조건 |
|------|------|------|
| ① | 필수 키워드 포함 | `배당`, `액티브`, `보험`, `은행` 중 하나 이상 |
| ② | 제외 키워드 필터 | `커버드콜`, `커버드`, `COVERED`, `CC`, `프리미엄`, `PREMIUM`, `인컴`, `INCOME` |
| ③ | ETF 브랜드 확인 | `KODEX`, `TIGER`, `KBSTAR`, `ARIRANG`, `HANARO`, `ACE`, `SOL`, `TIMEFOLIO`, `TIME`, `PLUS`, `WOORI`, `마이티`, `KOSEF`, `히어로즈`, `KOACT`, `BNK`, `HK`, `NAVIGATOR`, `파워`, `TREX` |

→ 일반 주식 혼입을 방지하기 위해 ③ ETF 브랜드명 포함 여부를 최종 검증

#### 2단계: ETF 현재가 조회 (청크 병렬)

```
API: getEtfPrice(symbol)
KIS TR_ID: FHPST02400000
URL: /uapi/etfetn/v1/quotations/inquire-price
```

- **15개씩 `Promise.all`로 병렬 호출** (KIS API 초당 20건 제한 준수)
- 청크 간 200ms 대기 (Vercel 60초 타임아웃 방지)
- 유효 데이터: `stck_prpr > 0`인 ETF만 후보로 유지
- 부가 정보: `etf_dvdn_cycl` (배당 주기 — 월배당/분기/반기/연)

#### 3단계: 배당 이력 조회 및 TOP10 선정

```
API: getKsdinfoDividend(options)
KIS TR_ID: HHKDB669102C0
URL: /uapi/domestic-stock/v1/ksdinfo/dividend
```

- 최근 2년 범위에서 해당 ETF의 실제 배당 지급 내역 조회
- `per_sto_divi_amt > 0` 필터 → `record_date` 내림차순 → 최신 배당 선택
- 15개씩 청크 병렬 처리

**수익률 및 부가 계산:**
```
수익률(%) = (주당배당금 / 현재 종가) × 100
횟수 = etf_dvdn_cycl 기반 추정 (월→12회, 분기→4회, 반기→2회, 연→1회)
가상배당금 = floor(1,000만원 / 종가) × 주당배당금
```

→ **수익률 내림차순 정렬 후 TOP 10** 추출

#### 4단계: 마크다운 리포트 생성

| 컬럼 | 내용 |
|------|------|
| 종목 | ETF 전체 명칭 |
| 종가 | 현재 종가 (원) |
| 주당배당금 | 실제 지급 금액 + 지급일 |
| 수익률 | 종가 대비 배당수익률 (%) |
| 횟수 | 연간 배당 지급 횟수 |
| 최근배당일 | 기준일(record_date) |
| 가상배당금 | 1,000만원 투자 시 예상 배당금 |

## 4. 사용된 KIS API 정리

| API 함수 | TR_ID | URL | 용도 |
|---------|-------|-----|------|
| `getDividendRateRanking` | `HHKDB13470100` | `/uapi/domestic-stock/v1/ranking/dividend-rate` | 배당률 상위 순위 조회 |
| `getKsdinfoDividend` | `HHKDB669102C0` | `/uapi/domestic-stock/v1/ksdinfo/dividend` | 예탁원 배당일정/이력 조회 |
| `getEtfPrice` | `FHPST02400000` | `/uapi/etfetn/v1/quotations/inquire-price` | ETF/ETN 현재가 조회 |
| `getDomesticPrice` | `FHKST01010100` | `/uapi/domestic-stock/v1/quotations/inquire-price` | 국내주식 현재가 조회 |
| `getStockInfo` | `CTPF1002R` | `/uapi/domestic-stock/v1/quotations/search-stock-info` | 종목 기본정보(한글명) |

## 6. 데이터 저장 구조

### Supabase 테이블: `study_boards`

| 컬럼 | 타입 | 설명 |
|------|------|------|
| `id` | UUID | 자동 생성 |
| `topic` | text | `'dividend'` (배당주/ETF 공통) |
| `title` | text | `배당주식_2026-03-24 16:52` 또는 `배당ETF_2026-03-24 16:52` |
| `content` | text | 생성된 마크다운 전문 |
| `created_at` | timestamp | 자동 생성 |
